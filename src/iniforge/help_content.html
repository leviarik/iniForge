iniForge Quick Start Guide

Getting Started

1. Set Working Directory:
• Enter folder path containing the configuration files
• Use Browse button or paste from clipboard
• Press Enter to load files

File Management

Filter Files:
• Use filename filter for quick searching
• Supports regex patterns (e.g., "config.*")
• Double-click file to open in external Meld editor
• Requires Meld tool to be installed

Search & Replace Operations

Replace Content Tab:
• Filter by content: Enter text to find
• Filter by regex: Use regex to find patterns
• Replace content with: Enter replacement text
• Click "Apply Content Replacements"

Add Configuration Tab:
• Select section from dropdown
• Enter configuration lines
• Choose to add at section start or end
• Click "Add Configuration"

Remove Configuration Tab:
• Enter content to remove in "Filter by content"
• Click "Remove Configuration"
• WARNING: This action cannot be undone!

File Editor

View & Edit:
• Click any file to view content
• Edit text directly in the editor
• Use search box to find text in current file
• Navigate between search matches with ← →

Search Features:
• Enter search term and press Enter
• Use "Case Sensitive" push button
• Navigate matches with Previous/Next buttons
• Clear highlights with clear button

Keyboard Shortcuts

• Ctrl+Delete: Clear text in any text field
• Enter: Search in editor / Load directory
• Double-click: Open file in external editor

Tips

• Best to use on local git repositories for version control benefits
• Always backup files before bulk operations
• Test operations on single files first
• Use file filtering to work with specific files only
• Hover over buttons for detailed tooltips

Interface

• Toggle dark/light theme with switch
• All text fields have clear buttons when needed
• Icons provide visual cues for actions
